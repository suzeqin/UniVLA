## 1. 项目概述

**UniVLA** (Universal Vision-Language-Action) 是一个基于任务中心潜在动作（Task-centric Latent Actions）的通用机器人策略模型。核心创新点：

- **统一动作空间**：通过潜在动作学习，在不同机器人形态间建立统一的、与具身无关的动作空间
- **两阶段训练**：
  - Stage 1: 学习任务无关的潜在动作
  - Stage 2: 学习任务中心的潜在动作
- **高效预训练**：仅需约960 A100 GPU小时，是OpenVLA计算资源的5%

### 3.1 潜在动作模型训练

#### Stage 1: 任务无关潜在动作 (`latent_action_model/genie/model.py`)

```python
from os import listdir, makedirs, path
from typing import Callable, Dict, Iterable, Tuple

import matplotlib.pyplot as plt
import numpy as np
import piq
import torch
import wandb
from PIL import Image
from einops import rearrange
from lightning import LightningModule  # 使用PyTorch Lightning进行训练
from torch import Tensor
from torch.optim import AdamW, Optimizer
from accelerate import PartialState

# 类型定义
OptimizerCallable = Callable[[Iterable], Optimizer]

# 导入潜在动作模型模块
from genie.modules import UncontrolledDINOLatentActionModel, ControllableDINOLatentActionModel
import logging
logging.basicConfig(format='%(message)s', level=logging.INFO)


class DINO_LAM(LightningModule):
    """
    基于DINO特征空间的潜在动作模型
    
    该模型在DINO编码的视觉特征空间中学习潜在动作表示，
    使用VQ-VAE进行离散化编码
    """

    def __init__(
            self,
            image_channels: int = 3,           # 输入图像通道数
            # 潜在动作模型参数
            lam_model_dim: int = 512,          # 模型维度
            lam_latent_dim: int = 32,          # 潜在空间维度
            lam_num_latents: int = 8,          # 潜在动作码本大小
            lam_patch_size: int = 16,          # 图像patch大小
            lam_enc_blocks: int = 8,           # 编码器层数
            lam_dec_blocks: int = 8,           # 解码器层数
            lam_num_heads: int = 8,            # 注意力头数
            lam_dropout: float = 0.0,          # Dropout率
            vq_beta: float = 0.25,             # VQ损失权重
            log_interval: int = 1000,          # 日志间隔
            log_path: str = "log_imgs",        # 日志路径
            task_name: str = 'lam_openx',      # 任务名称
            stage: str = 'stage-1',            # 训练阶段
            optimizer: OptimizerCallable = AdamW,  # 优化器
            make_data_pair: bool = False,      # 是否生成数据对
            stage_one_ckpt: str = None,        # Stage 1检查点路径
    ) -> None:
        super(DINO_LAM, self).__init__()
        assert stage in ['stage-1', 'stage-2']  # 验证训练阶段

        # 根据阶段选择模型类型
        # Stage 1: 使用无控制的LAM学习任务无关动作
        # Stage 2: 使用可控的LAM学习任务中心动作
        lam = UncontrolledDINOLatentActionModel if stage == 'stage-1' else ControllableDINOLatentActionModel
        
        # 初始化潜在动作模型
        self.lam = lam(
                    in_dim=image_channels,
                    model_dim=lam_model_dim,
                    latent_dim=lam_latent_dim,
                    num_latents=lam_num_latents,
                    patch_size=lam_patch_size,
                    enc_blocks=lam_enc_blocks,
                    dec_blocks=lam_dec_blocks,
                    num_heads=lam_num_heads,
                    dropout=lam_dropout,
                )
        
        # 如果是Stage 2，加载Stage 1的检查点
        if stage_one_ckpt and path.exists(stage_one_ckpt):
            lam_ckpt = torch.load(stage_one_ckpt)['state_dict']
            stage1_ckpt = {}
            for key in lam_ckpt.keys():
                # 只加载VQ和动作潜在表示相关的权重
                if 'vq' in key or 'action_latent' in key:
                    stage1_ckpt[key.replace("lam.", "")] = lam_ckpt[key]
            self.lam.load_state_dict(stage1_ckpt, strict=False)

        # 保存超参数
        self.lam_num_latents = lam_num_latents
        self.vq_beta = vq_beta
        self.log_interval = log_interval
        self.log_path = log_path
        self.optimizer = optimizer
        self.make_data_pair = make_data_pair

        self.save_hyperparameters()

        # 初始化WandB
        self.task_name = task_name
        self.distributed_state = PartialState()
        if self.distributed_state.is_main_process:
            wandb.init(name=task_name, reinit=True)

    def shared_step(self, batch: Dict) -> Tuple:
        """
        共享的训练/验证步骤
        
        Args:
            batch: 包含'videos', 'task_instruction', 'action', 'dataset_names'的字典
            
        Returns:
            outputs: 模型输出
            loss: 总损失
            loss_logs: 损失分量元组
        """
        # 前向传播
        outputs = self.lam(batch)
        gt_future_frames = outputs["target"]  # 真实未来帧

        # 计算损失
        # 1. MSE重建损失：衡量预测帧与真实帧的差异
        mse_loss = ((gt_future_frames - outputs["recon"]) ** 2).mean()
        
        # 2. VQ损失：使编码向量接近码本向量
        q_loss = ((outputs["emb"].detach() - outputs["z"]) ** 2).mean()
        
        # 3. 承诺损失：使码本向量接近编码向量
        commit_loss = ((outputs["emb"] - outputs["z"].detach()) ** 2).mean()

        # 总损失
        loss = mse_loss + q_loss + self.vq_beta * commit_loss
        
        # Stage 2额外优化不可控查询（码本保持冻结）
        if "z_q_uncontrol" in outputs.keys():
            q_loss_uncontrol = ((outputs["emb_uncontrol"].detach() - outputs["z_uncontrol"]) ** 2).mean()
            commit_loss_uncontrol = ((outputs["emb_uncontrol"]- outputs["z_uncontrol"].detach()) ** 2).mean()
            loss = loss + q_loss_uncontrol + self.vq_beta * commit_loss_uncontrol

        # 计算码本使用率
        unique, counts = torch.unique(outputs["indices"], return_counts=True)
        index_counts = torch.zeros(self.lam_num_latents, dtype=torch.long).cuda()
        index_counts[unique] = counts
        code_usage = (index_counts != 0).float().mean()

        # 构建损失日志
        loss_logs = (
            ("mse_loss", mse_loss),
            ("q_loss", q_loss),
            ("commit_loss", commit_loss),
            ("code_usage", code_usage),
        )

        # Stage 2额外的损失日志
        if "indices_uncontrol" in outputs.keys():
            unique, counts = torch.unique(outputs["indices_uncontrol"], return_counts=True)
            index_counts = torch.zeros(32, dtype=torch.long).cuda()
            index_counts[unique] = counts
            uncontrol_code_usage = (index_counts != 0).float().mean()

            loss_logs = (
                ("mse_loss", mse_loss),
                ("q_loss", q_loss),
                ("commit_loss", commit_loss),
                ("q_loss_uncontrol", q_loss_uncontrol),
                ("commit_loss_uncontrol", commit_loss_uncontrol),
                ("code_usage", code_usage),
                ("code_usage_uncontrol", uncontrol_code_usage),
            )

        return outputs, loss, loss_logs

    def training_step(self, batch: Dict, batch_idx: int) -> Tensor:
        """训练步骤"""
        outputs, loss, aux_losses = self.shared_step(batch)

        # 记录训练损失
        self.log_dict(
            {**{"train_loss": loss}, **{f"train/{k}": v for k, v in aux_losses}},
            prog_bar=True,
            logger=True,
            on_step=True,
            on_epoch=True,
            sync_dist=True  # 分布式同步
        )

        # WandB日志
        if self.distributed_state.is_main_process:
            wandb.log({**{"train_loss": loss}, **{f"train/{k}": v for k, v in aux_losses}})

        return loss

    @torch.no_grad()
    def test_step(self, batch: Dict, batch_idx: int) -> Tensor:
        """测试步骤（无梯度）"""
        outputs, loss, aux_losses = self.shared_step(batch)

        self.log_dict(
            {**{"test_loss": loss}, **{f"test/{k}": v for k, v in aux_losses}},
            prog_bar=True,
            logger=True,
            on_step=True,
            on_epoch=True,
            sync_dist=True
        )

        return loss

    def on_train_epoch_end(self):
        """每个训练epoch结束时的回调"""
        self.lam.vq.random_restart()  # VQ码本随机重启
        self.lam.vq.reset_usage()     # 重置码本使用统计

    def configure_optimizers(self) -> Optimizer:
        """配置优化器"""
        optim = self.optimizer(self.parameters())
        return optim
```

#### 潜在动作模块 (`latent_action_model/genie/modules/lam.py`)

```python
from typing import Dict, List

import torch
import torch.nn as nn
import torch.nn.functional as F
from torch import Tensor
from einops import rearrange, repeat
from transformers import T5EncoderModel, T5Tokenizer

from latent_action_model.genie.modules.blocks import (
    patchify, unpatchify, 
    SpatioTemporalTransformer, SpatioTransformer, 
    VectorQuantizer,
    MVSpatioTemporalTransformer, MVSpatioTransformer
)

from torchvision import transforms
# ImageNet标准化参数
IMAGENET_DEFAULT_MEAN = (0.485, 0.456, 0.406)
IMAGENET_DEFAULT_STD = (0.229, 0.224, 0.225)


class UncontrolledDINOLatentActionModel(nn.Module):
    """
    Stage 1: 无控制的潜在动作VQ-VAE
    
    学习任务无关的潜在动作表示，用于捕获通用的动作模式
    """

    def __init__(
            self,
            in_dim: int,                # 输入维度（图像通道数）
            model_dim: int,             # 模型隐藏维度
            latent_dim: int,            # 潜在空间维度
            num_latents: int,           # 码本大小
            patch_size: int,            # 图像patch大小
            enc_blocks: int,            # 编码器层数
            dec_blocks: int,            # 解码器层数
            num_heads: int,             # 注意力头数
            dropout: float = 0.0        # Dropout率
    ) -> None:
        super(UncontrolledDINOLatentActionModel, self).__init__()
        self.latent_dim = latent_dim
        self.patch_size = patch_size

        # DINO图像预处理和编码器
        self.dino_transform = transforms.Normalize(mean=IMAGENET_DEFAULT_MEAN, std=IMAGENET_DEFAULT_STD)
        self.dino_encoder = torch.hub.load('facebookresearch/dinov2', 'dinov2_vitb14_reg')
        self.dino_encoder.requires_grad_(False)  # 冻结DINO编码器

        dino_dim = 768  # DINOv2-Base的特征维度

        # 动作潜在表示（可学习参数）
        self.num_codes = 4  # 每帧的动作码数量
        self.action_latent = nn.Parameter(torch.empty(1, 1, self.num_codes, dino_dim))
        nn.init.uniform_(self.action_latent, a=-1, b=1)
        
        # 编码器：时空Transformer
        self.encoder = SpatioTemporalTransformer(
            in_dim=dino_dim,
            model_dim=model_dim,
            out_dim=latent_dim,
            num_blocks=enc_blocks,
            num_heads=num_heads,
            dropout=dropout,
            causal_temporal=True,  # 因果时间注意力
            to_out=False,
        )

        # VQ层：将连续潜在向量映射到离散码本
        self.to_codebook = nn.Linear(model_dim, latent_dim)
        self.vq = VectorQuantizer(
            num_latents=num_latents,
            latent_dim=latent_dim,
            code_restart=True,  # 启用码本重启机制
        )
        
        # 解码器
        self.patch_up = nn.Linear(dino_dim, model_dim)
        self.action_up = nn.Linear(latent_dim, model_dim)
        self.decoder = SpatioTransformer(
            in_dim=model_dim,
            model_dim=model_dim,
            out_dim=dino_dim,
            num_blocks=dec_blocks,
            num_heads=num_heads,
            dropout=dropout,
        )

        # T5文本编码器（用于语言条件）
        self.text_encoder = T5EncoderModel.from_pretrained('./t5-base')
        self.text_encoder.requires_grad_(False)  # 冻结文本编码器
        self.lang_proj = nn.Linear(768, model_dim)

        self.tokenizer = T5Tokenizer.from_pretrained('./t5-base')

    def encode_text(self, lang: List):
        """编码文本指令"""
        encoding = self.tokenizer(lang, return_tensors="pt", padding=True).to(self.device)
        input_ids = encoding['input_ids']
        attention_mask = encoding['attention_mask']

        with torch.no_grad():
            encoder_outputs = self.text_encoder(input_ids=input_ids, attention_mask=attention_mask)

        last_hidden_states = encoder_outputs.last_hidden_state
        return last_hidden_states, attention_mask

    def vq_encode(self, videos: Tensor, lang_embed: Tensor = None, attention_mask: Tensor = None) -> Dict:
        """
        VQ编码：将视频序列编码为离散潜在动作
        
        Args:
            videos: 视频张量 [B, T, C, H, W]
            lang_embed: 语言嵌入（可选）
            attention_mask: 注意力掩码（可选）
        
        Returns:
            包含patches, z_q, z, emb, indices的字典
        """
        B, T = videos.shape[:2]
        
        # 使用DINO编码视频帧
        videos = rearrange(videos, "b T c h w -> (b T) c h w")
        videos = self.dino_transform(videos)
        dion_features = self.dino_encoder.forward_features(videos)['x_norm_patchtokens']
        dion_features = rearrange(dion_features, "(b T) l d -> b T l d", T=2)

        # 添加动作潜在表示
        action_pad = self.action_latent.expand(B, T, -1, -1)
        padded_patches = torch.cat([action_pad, dion_features], dim=2)

        # 编码
        z = self.encoder(padded_patches, lang_embed, attention_mask)
      
        # 获取未来帧的潜在动作
        z = self.to_codebook(z[:, 1:, :self.num_codes])  # (B, T-1, n, E)

        # 向量量化
        z = z.reshape(B * (T - 1), self.num_codes, self.latent_dim)
        z_q, z, emb, indices = self.vq(z)
        z_q = z_q.reshape(B, T - 1, self.num_codes, self.latent_dim)
        
        return {
            "patches": dion_features,
            "z_q": z_q,
            "z": z,
            "emb": emb,
            "indices": indices,
        }

    def forward(self, batch: Dict) -> Dict:
        """
        前向传播
        
        Args:
            batch: 包含videos和task_instruction的批次数据
        
        Returns:
            包含重建结果和目标的输出字典
        """
        B, T = batch["videos"].shape[:2]
        H, W = batch["videos"].shape[3:5]

        # 编码文本指令
        lang_embed, attention_mask = self.encode_text(batch["task_instruction"])
        lang_embed = self.lang_proj(lang_embed)
        attention_mask = torch.cat([
            torch.ones((B, self.num_codes + (H // self.patch_size)**2)).to(self.device),
            attention_mask
        ], dim=-1)

        # VQ编码
        outputs = self.vq_encode(
            batch["videos"], 
            repeat(lang_embed, 'b l d -> b T l d', T=T), 
            attention_mask.repeat(T, 1)
        )
        
        # 准备解码输入
        video_patches = self.patch_up(outputs["patches"][:, :-1])
        action_patches = self.action_up(outputs["z_q"])
        video_action_patches = torch.cat([action_patches, video_patches], dim=2)

        # 解码
        video_recon = self.decoder(video_action_patches, lang_embed.unsqueeze(1), attention_mask)
        video_recon = video_recon[:, :, self.num_codes: self.num_codes + video_patches.shape[2]]

        outputs.update({
            "recon": video_recon,                    # 重建的视频特征
            "target": outputs["patches"][:, [-1]]   # 目标帧特征
        })
        return outputs

    @property
    def device(self):
        return next(self.parameters()).device


class ControllableDINOLatentActionModel(nn.Module):
    """
    Stage 2: 可控的潜在动作VQ-VAE
    
    在Stage 1的基础上学习任务中心的潜在动作表示
    同时保留任务无关的动作表示用于处理背景变化
    """

    def __init__(
            self,
            in_dim: int,
            model_dim: int,
            latent_dim: int,
            num_latents: int,
            patch_size: int,
            enc_blocks: int,
            dec_blocks: int,
            num_heads: int,
            dropout: float = 0.0
    ) -> None:
        super(ControllableDINOLatentActionModel, self).__init__()
        self.latent_dim = latent_dim
        self.patch_size = patch_size

        # DINO编码器（与Stage 1相同）
        self.dino_transform = transforms.Normalize(mean=IMAGENET_DEFAULT_MEAN, std=IMAGENET_DEFAULT_STD)
        self.dino_encoder = torch.hub.load('facebookresearch/dinov2', 'dinov2_vitb14_reg')
        self.dino_encoder.requires_grad_(False)

        dino_dim = 768

        # 任务无关的动作潜在表示（从Stage 1继承）
        self.num_codes = 4
        self.action_latent = nn.Parameter(torch.empty(1, 1, self.num_codes, dino_dim))
        nn.init.uniform_(self.action_latent, a=-1, b=1)
        
        # 编码器
        self.encoder = SpatioTemporalTransformer(
            in_dim=dino_dim,
            model_dim=model_dim,
            out_dim=latent_dim,
            num_blocks=enc_blocks,
            num_heads=num_heads,
            dropout=dropout,
            causal_temporal=True,
            to_out=False,
        )

        # 两个VQ层：
        # 1. vq: 任务无关的码本（从Stage 1加载并冻结）
        # 2. vq_action: 任务中心的码本（Stage 2训练）
        self.to_codebook = nn.Linear(model_dim, latent_dim)
        self.to_codebook_uncontrol = nn.Linear(model_dim, latent_dim)
        
        self.vq = VectorQuantizer(
            num_latents=16,  # 任务无关码本大小
            latent_dim=latent_dim,
            code_restart=True,
        )
        
        # 解码器
        self.patch_up = nn.Linear(dino_dim, model_dim)
        self.action_up = nn.Linear(latent_dim, model_dim)
        self.action_up_uncontrol = nn.Linear(latent_dim, model_dim)
        self.decoder = SpatioTransformer(
            in_dim=model_dim,
            model_dim=model_dim,
            out_dim=dino_dim,
            num_blocks=dec_blocks,
            num_heads=num_heads,
            dropout=dropout,
        )

        # 任务中心的动作码本
        self.vq_action = VectorQuantizer(
            num_latents=num_latents,  # 任务中心码本大小
            latent_dim=latent_dim,
            code_restart=True,
        )
        
        # 任务中心的动作潜在表示
        self.action_latent_controllable = nn.Parameter(torch.empty(1, 1, self.num_codes, dino_dim))
        nn.init.uniform_(self.action_latent_controllable, a=-1, b=1)

        # 冻结任务无关的VQ码本
        self.vq.requires_grad_(False)

    def vq_encode(self, videos: Tensor, lang_embed: Tensor = None, attention_mask: Tensor = None) -> Dict:
        """VQ编码（包含任务中心和任务无关两种）"""
        B, T = videos.shape[:2]
        
        # DINO编码
        videos = rearrange(videos, "b T c h w -> (b T) c h w")
        videos = self.dino_transform(videos)
        dion_features = self.dino_encoder.forward_features(videos)['x_norm_patchtokens']
        dion_features = rearrange(dion_features, "(b T) l d -> b T l d", T=2)

        # 拼接任务无关和任务中心的动作潜在表示
        action_pad = self.action_latent.expand(B, T, -1, -1)
        padded_patches = torch.cat([action_pad, dion_features], dim=2)
        action_pad_controllable = self.action_latent_controllable.expand(B, T, -1, -1)
        padded_patches = torch.cat([action_pad_controllable, padded_patches], dim=2)

        # 编码
        z = self.encoder(padded_patches)
      
        # 任务无关潜在动作
        z_uncontrol = self.to_codebook_uncontrol(z[:, 1:, self.num_codes : self.num_codes * 2])
        z_uncontrol = z_uncontrol.reshape(B * (T - 1), self.num_codes, self.latent_dim)
        z_q_uncontrol, z_uncontrol, emb_uncontrol, indices_uncontrol = self.vq(z_uncontrol)
        z_q_uncontrol = z_q_uncontrol.reshape(B, T - 1, self.num_codes, self.latent_dim)

        # 任务中心潜在动作
        z_action = self.to_codebook(z[:, 1:, :self.num_codes])
        z_action = z_action.reshape(B * (T - 1), self.num_codes, self.latent_dim)
        z_q, z, emb, indices = self.vq_action(z_action)
        z_q = z_q.reshape(B, T - 1, self.num_codes, self.latent_dim)

        return {
            "patches": dion_features,
            "z_q": z_q,
            "z": z,
            "emb": emb,
            "z_q_uncontrol": z_q_uncontrol,
            "z_uncontrol": z_uncontrol,
            "emb_uncontrol": emb_uncontrol,
            "indices": indices,
            "indices_uncontrol": indices_uncontrol,
        }

    def forward(self, batch: Dict) -> Dict:
        """前向传播"""
        B, T = batch["videos"].shape[:2]
        H, W = batch["videos"].shape[3:5]

        outputs = self.vq_encode(batch["videos"])
        video_patches = self.patch_up(outputs["patches"][:, :-1])

        # 解码：结合任务中心和任务无关的动作表示
        video_action_patches = torch.cat([
            self.action_up(outputs["z_q"]),              # 任务中心动作
            self.action_up_uncontrol(outputs["z_q_uncontrol"]),  # 任务无关动作
            video_patches
        ], dim=2)
        
        video_recon = self.decoder(video_action_patches)
        video_recon = video_recon[:, :, -video_patches.shape[2]:]

        outputs.update({
            "recon": video_recon,
            "target": outputs["patches"][:, [-1]]
        })
        return outputs

    @property
    def device(self):
        return next(self.parameters()).device
```

### 3.2 VLA预训练 (`vla-scripts/train.py`)

```python
import json
import os
import re
from dataclasses import dataclass, field
from pathlib import Path
from typing import Optional, Tuple, Union

import draccus                           # 配置管理库
import torch
import torch.distributed as dist
import torchvision.transforms as transforms
import yaml

from prismatic.conf import VLAConfig, VLARegistry
from prismatic.models import load, load_vla
from prismatic.overwatch import initialize_overwatch
from prismatic.training import VLAMetrics, get_train_strategy
from prismatic.util import set_global_seed
from prismatic.vla import get_latent_vla_dataset_and_collator
from prismatic.vla.datasets.rlds.utils.data_utils import save_dataset_statistics

# 禁用分词器并行
os.environ["TOKENIZERS_PARALLELISM"] = "false"

# 初始化日志记录器
overwatch = initialize_overwatch(__name__)


@dataclass
class TrainConfig:
    """训练配置类"""
    
    # VLA配置
    vla: VLAConfig = field(
        default_factory=VLAConfig.get_choice_class(VLARegistry.DINOSIGLIP_224PX_MX_BRIDGE.vla_id)
    )
    pretrain_vlm: str = '/path/to/your/prism-dinosiglip-224px_7b'  # 预训练VLM路径
    lam_path: str = "latent_action_model/logs/task_centric_lam_stage2/epoch=0-step=200000.ckpt"  # LAM检查点

    # LAM超参数
    codebook_size: int = 16            # 码本大小
    lam_model_dim: int = 768           # LAM模型维度
    lam_latent_dim: int = 128          # 潜在空间维度
    lam_patch_size: int = 14           # 图像patch大小
    lam_enc_blocks: int = 12           # 编码器层数
    lam_dec_blocks: int = 12           # 解码器层数
    lam_num_heads: int = 12            # 注意力头数

    # 目录配置
    data_root_dir: Path = Path("/path/to/your/rlds_data_collection")  # 数据目录
    run_root_dir: Path = Path("runs")                                  # 日志目录

    # 恢复训练参数
    pretrained_checkpoint: Optional[Path] = None
    is_resume: bool = True
    resume_step: Optional[int] = None
    resume_epoch: Optional[int] = None

    # 训练参数
    run_id: Optional[str] = None
    run_id_note: Optional[str] = None
    save_interval: int = 10000          # 检查点保存间隔
    image_aug: bool = True              # 是否启用图像增强
    seed: int = 42

    # HuggingFace Hub凭证
    hf_token: Union[str, Path] = ''

    # 跟踪参数
    trackers: Tuple[str, ...] = ("jsonl", "wandb")
    wandb_project: str = "latent-action-pretrain"
    wandb_entity: str = "opendrivelab"

    def __post_init__(self) -> None:
        """从vla配置提取优化参数"""
        self.epochs = self.vla.epochs
        self.max_steps = self.vla.max_steps
        self.global_batch_size = self.vla.global_batch_size
        self.per_device_batch_size = self.vla.per_device_batch_size
        self.learning_rate = self.vla.learning_rate
        self.weight_decay = self.vla.weight_decay
        self.max_grad_norm = self.vla.max_grad_norm
        self.lr_scheduler_type = self.vla.lr_scheduler_type
        self.warmup_ratio = self.vla.warmup_ratio
        self.train_strategy = self.vla.train_strategy

        # 验证world_size
        assert (
            self.vla.expected_world_size == overwatch.world_size()
        ), f"Expected World Size = {self.vla.expected_world_size} but Found {overwatch.world_size()} GPUs!"


@draccus.wrap()
def train(cfg: TrainConfig) -> None:
    """VLA预训练主函数"""
    
    overwatch.info("OpenVLA Training :: Warming Up")

    # 设置CUDA设备
    torch.cuda.set_device(device_id := overwatch.local_rank())
    torch.cuda.empty_cache()

    # 配置运行ID
    vla_id = cfg.vla.vla_id
    cfg.run_id = (
        f"{vla_id}+n{cfg.vla.expected_world_size // 8}+b{cfg.per_device_batch_size}+x{cfg.seed}"
        if cfg.run_id is None
        else cfg.run_id
    )
    if cfg.run_id_note is not None:
        cfg.run_id += f"--{cfg.run_id_note}"
    if cfg.image_aug:
        cfg.run_id += "--image_aug"
    cfg.run_id += '-Latent-Action-Pretraining'

    # 创建目录
    hf_token = cfg.hf_token
    worker_init_fn = set_global_seed(cfg.seed, get_worker_init_fn=True)
    os.makedirs(run_dir := (cfg.run_root_dir / cfg.run_id), exist_ok=True)
    os.makedirs(cfg.run_root_dir / cfg.run_id / "checkpoints", exist_ok=True)

    # 保存配置
    if overwatch.is_rank_zero():
        draccus.dump(cfg, open(run_dir / "config.yaml", "w"))
        with open(run_dir / "config.yaml", "r") as f_yaml, open(run_dir / "config.json", "w") as f_json:
            yaml_cfg = yaml.safe_load(f_yaml)
            json.dump(yaml_cfg, f_json, indent=2)

    # 加载VLM模型
    overwatch.info(f"Loading Base VLM `{cfg.vla.base_vlm}` from ID/Path")
    if cfg.pretrained_checkpoint is not None:
        if cfg.is_resume:
            assert int(re.search("step-(.+?)-", cfg.pretrained_checkpoint.name).group(1)) == cfg.resume_step
            assert int(re.search("epoch-(.+?)-", cfg.pretrained_checkpoint.name).group(1)) == cfg.resume_epoch
        vlm = load_vla(cfg.pretrained_checkpoint, hf_token=hf_token, load_for_training=True, cache_dir=cfg.pretrain_vlm)
    else:
        vlm = load(cfg.pretrain_vlm, hf_token=hf_token, load_for_training=True, cache_dir=cfg.pretrain_vlm)

    # 验证模型精度
    for param in vlm.parameters():
        assert param.dtype == torch.float32, f"Loaded VLM parameter not in full precision: {param}"

    # 确定训练阶段
    if not cfg.vla.freeze_vision_backbone and not cfg.vla.freeze_llm_backbone:
        stage = "vla-full-train"
    elif cfg.vla.freeze_vision_backbone and not cfg.vla.freeze_llm_backbone:
        stage = "vla-train"
    elif not cfg.vla.freeze_vision_backbone and cfg.vla.freeze_llm_backbone:
        stage = "vla-sandwich-train"
    elif cfg.vla.freeze_vision_backbone and cfg.vla.freeze_llm_backbone:
        stage = "vla-last-layer-train"

    # 冻结骨干网络
    overwatch.info(f"Invoking `VLM.freeze_backbones()` for `{vla_id}` => Stage: `{stage}`")
    vlm.freeze_backbones(stage)

    # 打印参数统计
    num_params = sum(p.numel() for p in vlm.parameters())
    num_trainable_params = sum(p.numel() for p in vlm.parameters() if p.requires_grad)
    overwatch.info(
        f"# Parameters (in millions): {num_params / 10**6:.3f} Total, {num_trainable_params / 10**6:.3f} Trainable"
    )

    # 加载潜在动作模型
    from latent_action_model.genie.modules.lam import ControllableDINOLatentActionModel

    latent_action_model = ControllableDINOLatentActionModel(
        in_dim=3,
        model_dim=cfg.lam_model_dim,
        latent_dim=cfg.lam_latent_dim,
        num_latents=cfg.codebook_size,
        patch_size=cfg.lam_patch_size,
        enc_blocks=cfg.lam_enc_blocks,
        dec_blocks=cfg.lam_dec_blocks,
        num_heads=cfg.lam_num_heads,
        dropout=0.,
    )

    # 加载LAM检查点
    lam_ckpt = torch.load(cfg.lam_path)['state_dict']
    new_ckpt = {}
    for key in lam_ckpt.keys():
        new_ckpt[key.replace("lam.", "")] = lam_ckpt[key]
    latent_action_model.load_state_dict(new_ckpt, strict=True)
    latent_action_model = latent_action_model.to(device_id).eval()

    # 创建数据集
    overwatch.info(f"Creating VLA Open-X Dataset with Mixture `{cfg.vla.data_mix}`")
    vla_dataset, action_tokenizer, collator = get_latent_vla_dataset_and_collator(
        cfg.data_root_dir,
        cfg.vla.data_mix,
        image_transform=vlm.vision_backbone.get_image_transform(),
        image_transform_lam=transforms.ToTensor(),
        latent_action_tokenizer=latent_action_model,
        tokenizer=vlm.llm_backbone.get_tokenizer(),
        prompt_builder_fn=vlm.llm_backbone.prompt_builder_fn,
        default_image_resolution=vlm.vision_backbone.default_image_resolution,
        shuffle_buffer_size=cfg.vla.shuffle_buffer_size,
        image_aug=cfg.image_aug,
    )

    # 添加动作特殊token
    special_tokens_dict = {'additional_special_tokens': [f'<ACT_{i}>' for i in range(cfg.codebook_size)]}
    num_added_toks = action_tokenizer.add_special_tokens(special_tokens_dict)

    # 保存数据集统计
    if overwatch.is_rank_zero():
        save_dataset_statistics(vla_dataset.dataset_statistics, run_dir)

    # 创建训练策略
    overwatch.info(f"Initializing Train Strategy `{cfg.train_strategy}`")
    train_strategy = get_train_strategy(
        train_strategy=cfg.train_strategy,
        vlm=vlm,
        device_id=device_id,
        stage=stage,
        epochs=cfg.epochs,
        max_steps=cfg.max_steps,
        global_batch_size=cfg.global_batch_size,
        per_device_batch_size=cfg.per_device_batch_size,
        learning_rate=cfg.learning_rate,
        weight_decay=cfg.weight_decay,
        max_grad_norm=cfg.max_grad_norm,
        lr_scheduler_type=cfg.lr_scheduler_type,
        warmup_ratio=cfg.warmup_ratio,
        enable_gradient_checkpointing=cfg.vla.enable_gradient_checkpointing,
        enable_mixed_precision_training=cfg.vla.enable_mixed_precision_training,
        reduce_in_full_precision=cfg.vla.reduce_in_full_precision,
        worker_init_fn=worker_init_fn,
    )
    train_strategy.run_setup(run_dir=run_dir, n_train_examples=len(vla_dataset))

    # 创建指标跟踪器
    overwatch.info(f"Creating Metrics with Active Trackers => `{cfg.trackers}`")
    metrics = VLAMetrics(
        cfg.trackers,
        cfg.run_id,
        run_dir,
        draccus.encode(cfg),
        wandb_project=cfg.wandb_project,
        wandb_entity=cfg.wandb_entity,
        resume_step=cfg.resume_step,
        resume_epoch=cfg.resume_epoch,
    )

    # 开始训练
    overwatch.info("Starting VLA Latent Action Training Loop")
    train_strategy.run_latent_action_training(
        vla_dataset,
        collator,
        action_tokenizer,
        metrics,
        save_interval=cfg.save_interval,
    )

    # 完成训练
    overwatch.info("Done with Training =>> Finalizing Metrics")
    metrics.finalize()

    overwatch.info("... and that's all, folks!")
    dist.barrier()
    dist.destroy_process_group()


if __name__ == "__main__":
    train()
```

## 6. 与InternVLA-A1预训练的对比分析

### 6.1 架构对比

| 特性 | InternVLA-A1 | UniVLA |
|------|-------------|--------|
| 基础架构 | Qwen3-VL + 专家模型 | Prismatic VLM + LAM |
| 动作表示 | 连续动作 + Flow Matching | 离散潜在动作 + VQ-VAE |
| 视觉编码 | Qwen3-VL视觉编码器 + COSMOS | DINOv2 + SigLIP |
| 生成能力 | 视觉预见生成 | 潜在动作生成 |
| 训练框架 | HuggingFace Accelerate | torchrun + Lightning |

### 6.2 数据格式对比

| 特性 | InternVLA-A1 | UniVLA |
|------|-------------|--------|
| 数据格式 | LeRobot V3.0 (Parquet) | RLDS (TFRecord) |
| 视频存储 | MP4视频文件 | 图像帧 |
| 动作表示 | 连续向量 | 离散token |
| 统计信息 | stats.json | dataset_statistics.json |

### 6.3 训练策略对比

| 特性 | InternVLA-A1 | UniVLA |
|------|-------------|--------|
| 损失函数 | MSE (动作) + MSE (视觉) | Cross-Entropy + VQ损失 |
| 预训练阶段 | 单阶段联合训练 | 两阶段（LAM + VLA） |
| 分布式策略 | Accelerate DDP | torchrun DDP |
| 混合精度 | BFloat16 | FP16 |

### 6.4 各自优势

**InternVLA-A1 优势**：
1. **端到端训练**：视觉理解、预见生成和动作预测联合优化
2. **连续动作空间**：支持细粒度的动作控制
3. **视觉预见**：能够预测未来场景，增强对动态环境的适应性
4. **LeRobot生态**：与LeRobot框架深度集成，便于部署

**UniVLA 优势**：
1. **跨具身泛化**：通过潜在动作实现不同机器人形态间的迁移
2. **计算高效**：预训练计算资源仅为OpenVLA的5%
3. **人类数据利用**：可直接使用无动作标注的人类视频数据
4. **模块化设计**：LAM和VLA可独立训练和更新

### 6.5 适用场景

| 场景 | 推荐方案 |
|------|---------|
| 单一机器人高精度操作 | InternVLA-A1 |
| 多机器人跨平台部署 | UniVLA |
| 动态环境任务 | InternVLA-A1 |
| 资源受限场景 | UniVLA |
| 需要人类视频数据 | UniVLA |
